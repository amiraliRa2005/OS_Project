# OS_Project
Project Phases
Phase 1: System Calls & User Utilities
Implemented essential user-level utilities and custom system calls (such as date, trace, and find) to extend the functionality and interactivity of the XV6 operating system.
Phase 2: Process Scheduling & Memory Management
Enhanced the XV6 kernel by implementing custom scheduling algorithms (such as Lottery Scheduling) and modifying memory management mechanisms to improve process efficiency.
Phase 3: Multithreading & Synchronization
Introduced native kernel-level thread support and implemented robust synchronization primitives (such as Mutexes and Condition Variables) to manage concurrent execution and prevent race conditions.
